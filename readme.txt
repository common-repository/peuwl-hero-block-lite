Highly customizable image hero block for WordPress 5+ (Gutenberg).

== Description ==


Peuwl Hero Block LITE is a simple-to-use Hero Section Plugin. 

Justlook inside the "Peuwl Blocks" category in the block inserter to display the Hero, then configurate it how ever you want.

This plugin is a free version of "Peuwl Hero Block", which means it got some limitations. You will only be able to use the image hero setting.

**Please Note:** This plugin requires WordPress 5.0+ (the Gutenberg editor) in order for it to work.

== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to _Plugins &rarr; Add New.
2. Type "Peuwl Hero Block LITE" into the Search and hit Enter.
3. Locate the plugin in the list of search results and click **Install Now**.
4. Once installed, click the **Activate** link.
5. BAM - now you will find the block inside the block inserter in your editor under the category "Peuwl Blocks".

It can also be installed manually.

1. Download the "Peuwl Hero Block LITE" plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate Product Blocks in the list and click the **Activate** link.
5. You will now find the block inside the block inserter in your editor under the category "Peuwl Blocks".

== Frequently Asked Questions ==

= Can Peuwl Hero Block LITE be used with any theme? =

There are some themes that are not quite ready for the Gutenberg editor yet. But download the plugin and test it out!
It's also preferable to use a full width template (no sidebars etc.) for it to work as intended.
